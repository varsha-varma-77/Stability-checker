# 🧠 Emotional Stability Checker

An AI-powered web application that predicts a student's emotional stability and stress levels based on their daily lifestyle habits.

## 📌 Project Overview
This project uses a dataset of 2,000 students to analyze the relationship between academic workload, physical health, and emotional well-being. By inputting daily habits like sleep and study hours, the app provides a real-time stability assessment.

## 🛠️ Tech Stack
- **Language:** Python
- **Framework:** Streamlit (Web UI)
- **Machine Learning:** Scikit-Learn (Random Forest Classifier)
- **Data Handling:** Pandas & NumPy

## 📊 Dataset Features
The model evaluates stability based on:
* **Study Hours:** Daily time spent on academics.
* **Sleep Hours:** Quality of rest.
* **Physical Activity:** Hours of exercise.
* **Social/Extracurricular:** Balance between work and life.
* **GPA:** Academic performance metric.

## 🚀 How to Run Locally
If you have Python installed, follow these steps:
1. Clone the repo: `git clone https://github.com/your-username/stability-checker.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`
